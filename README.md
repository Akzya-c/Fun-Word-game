# Word Scramble Puzzle Game

A fun and beginner-friendly Python puzzle game where players must guess the correct word from scrambled letters within limited attempts.

This project is designed to help beginners practice core Python concepts such as loops, functions, conditionals, lists, and randomization.


## How the Game Works

1. The program randomly selects a word.
2. The letters of the word are scrambled.
3. The player tries to guess the original word.
4. The player has limited attempts to win the game.



## Technologies Used

- Python 3
- Built-in random module



## How to Run the Game

1. Make sure Python is installed on your system.
2. Clone the repository or download the source file.
3. Run the following command in the terminal:

```bash
python word_scramble.py
